Coding style

To print pretty code on our pages, we can use Rainbow, a code syntax highlighting library written in Javascript.

It allows different language sintax and CSS themes.

To allow code sintax highlightning, use <code>...</code> or <pre>...</pre> tags, with a .data-language="LANGUAGE_NAME" nested to it. 


        
    1.  Rainbow.extend('css', [
    2.      {
    3.          'name': 'comment',
    4.          'pattern': /\/\*[\s\S]*?\*\//gm
    5.      },
    6.      {
    7.          'name': 'constant.hex-color',
    8.          pattern': /#([a-f0-9]{3}|[a-f0-9]{6})(?=;|\s)/gi
    9.      },
    10.     {
    11.         'matches': {
    12.             1: 'constant.numeric',
    13.             2: 'keyword.unit'
    14.         },
    15.         'pattern': /(\d+)(px|cm|s|%)?/g
    16.     }
    17. ], true);
